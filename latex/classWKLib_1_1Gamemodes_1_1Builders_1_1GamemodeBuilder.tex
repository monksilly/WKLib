\doxysubsection{Detailed Description}
A fluent builder class for creating and configuring an M\+\_\+\+Gamemode instance.~\newline
 It allows setting various properties of a gamemode such as regions, name, intro text, and gameplay flags in a chainable manner before finally building the M\+\_\+\+Gamemode object.
